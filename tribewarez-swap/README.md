# tribewarez-swap

PTtC token swap AMM (Automated Market Maker) for Tribewarez DeFi. Constant product formula (x * y = k), liquidity provision with LP tokens, 0.30% swap fee, and slippage protection.

Part of the [Tribewarez programs](../README.md) workspace. Build with `anchor build -p tribewarez_swap` from the workspace root.
