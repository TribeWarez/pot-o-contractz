# tribewarez-pot-o

PoT-O (Proof of Tensor Optimizations) on-chain Solana program for Tribewarez DeFi. Handles miner registration, proof submission and validation, reward distribution, and difficulty adjustment.

Part of the [Tribewarez programs](../README.md) workspace. Build with `anchor build -p tribewarez_pot_o` from the workspace root.
